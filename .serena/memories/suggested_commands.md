# Suggested Commands
- Install deps: `npm install`
- Run dev server with Vite middleware: `npm run dev` (respects `PORT`, default 3000).
- Type-check all code: `npm run check`
- Build client + bundle server to `dist/`: `npm run build`
- Serve built app: `npm run start`
- Push schema changes to Postgres via Drizzle: `npm run db:push`
- Inspect git history for conventions: `git log --oneline`
